MongoDB学习笔记（一，安装MongoDB&可视化工具Compose）

Windows（MongoDB）下的安装

1.首先从官网下载MongoDB的zip包，解压到指定的目录，在创建两个文件夹分别是MongoDBDATA,MongoDBLOG,这两个文件夹可以随意命名，分别用来存储数据和日志文件，为了见名知义，我用了上面的命名。在MongDBLOG这个文件夹下创建一个mongodb.log文件，这个文件的作用是用来保存MongoDB的日志文件。

2.接下来使用CMD，进入到MongDB的bin目录下，执行mongod –dpath:（D:MongoDB\MongoDBDATA）此处的路径与上面你创建的文件夹目录一致即可。

3.在MongoDB的安装路径下，创建mongod.cfg（采用了YAML的文件格式）, 使用SubLine 打开后编辑内容
systemLog: (指定MongoDB的log文件存储的位置)
destination: file
path: D:\MongoDB\MongoDBLOG\mongod.log
storage: （指定MongoDB的数据存储的位置）
journal:
enabled: true
dbPath: D:\MongoDB\MongoDBDATA
bind_ip: (指定MongoDB的绑定IP，此处指定为0.0.0.0为任意IP都可访问)
0.0.0.0
net:
port:27017(指定MongoDB的端口)
security:
authorization:enable(启用认证！)

4.设置新的环境变量MONGO_HOME，即MongoDB的安装目录，在PATH环境变量下加入%MONGO_HOME%\bin, 这一步的作用是让Mongo命令可以在任意盘符使用。

5.接下来安装MongoDB的服务，使其开机自启动，避免每次启动的麻烦，也可以省略这一步。

6.使用管理员身份打开CMD，切换到MongoDB的bin目录下，使用下面的命令

mongod –config D:\MongoDB\mongod.cfg –serviceName MongoDB –install

7.安装完成之后，可以通过Windows的服务管理器设置为开机自启动，就可以避免每次手动启动MongoDB的服务了。

Windows(MongDB Compose)可视化图形管理工具的安装

1.首先在MongoDB的官方网站上下载MongoDBCompose的安装包
https://www.mongodb.com/download-center?jmp=blog#compass

2.双击这个exe程序，完成安装即可。
